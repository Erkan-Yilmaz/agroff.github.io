QT, Social, and Electrum Wallets

Reddcoin currently offers 3 different wallets. Knowing which one(s) are best for you can be confusing. Luckily, we're here to help.

QT is the original Reddcoin wallet. Its roots trace back to the original bitcoin wallet written by Satoshi Nakamoto, and it is constantly being updated with the latest features from bitcoin.

The QT Wallet is recommended for tech savvy users or users who require the highest levels of security, stability, and features which help understand how Reddcoin works.

The Reddcoin Social Wallet is a new twist on the QT wallet. The Social Wallet uses the same code under the hood (it interfaces with the same daemon used by the QT wallet), but it includes a fresh new user experience including real time price conversion, twitter feed integration, and an embedded IRC Client.

The Social Wallet is recommended for users who aren't interested in the inner workings of Reddcoin, but just want a beautiful way to save and spend their Reddcoin.

Reddcoin Electrum is a fast, secure, and user friendly Reddcoin wallet. Electrum doesn't download the blockchain, so there's no more need to wait for your wallet to sync. Just open your wallet and start using it.

Electrum backups are simplified by a natural language passphrase which can be written down or memorized. With Electrum, a single backup will keep your coins safe forever.

Reddcoin Electrum is recommended for users who don't want the hassle of downloading the block chain, waiting for their wallet to sync, or making constant backups. It is especially good for user's with lower capacity computers as it uses less resources than traditional full node wallets.

Electrum is able to do all of this because it's a completely different class of wallet, called an SPV Wallet.

SPV vs Full Node

Until the release of Reddcoin Electrum, full node wallets the only ones available. A full node wallet downloads the enter blockchain (the public ledger of all transactions that have ever occurred) and uses that ledger to ensure transactions are valid. Since a full node knows about every transaction that has ever been spent, it can quickly reject transactions which attempt to spend coins they don't own any more.

SPV, or Simplified Payment Verification, can verify transactions without downloading the entire blockchain. SPV works by downloading only block headers (the part of a block which doesn't contain any transactions) and figuring out what blocks relevant transaction occurred in. Once an SPV wallet knows where in the blockchain a transaction occured, they can verify the transaction by seeing that more valid blocks come after the block that includes the transaction. Instead of independently verifying the transactions, SPV trusts that other full nodes in the network have verified it once it is deep enough in the blockchain (AKA, it has enough confirmations).

While full nodes also rely on confirmations, they do it with a different purpose. A full node verifies the transaction independently and then uses confirmations to determine the likelihood of a competing fork erasing the subchain which contains the transaction. SPV uses the confirmation themselves to verify that the transaction was valid.

Electrum Features

  • Fast: Your wallet doesn't download the blockchain, so it's ready to use instantly.
  • Forgiving: Your wallet can be easily recovered from your natural language passphrase.
  • Safe: Your private keys don't leave your computer and server data is verified using SPV.
  • Auto Sync: Electrum wallets on all devices stay in sync automatically.
  • Cold Storage: You can have secure offline wallets and still safely spend from an online computer.
  • Compatible: Your private keys can be used with other Reddcoin wallets.

Electrum FAQ

What is the Seed?
Reddcoin Electrum uses a 128-bit random seed to generate your private keys. The seed can be represented as a 12-word pass-phrase which is easy to write down or remember. Regular backups are no longer necessary because all of your addresses can be recovered from a single pass-phrase.
What is the gap limit?
The gap limit is the maximum number of consecutive unused addresses in your deterministic sequence of addresses. Reddcoin Electrum uses a gap limit to stop looking for addresses. This is set to 5 by default, so the client generates new addresses until 5 unused addresses are found.
What does it mean to "freeze" an address?
When you freeze an address, the funds in that address will not be used for sending Reddcoin. You cannot send Reddcoin if you don't have enough funds in non-frozen addresses.
What should I do with my old Reddcoin addresses when switching to Reddcoin Electrum?
The best way to switch to Electrum is to send all your Reddcoin to one of the your Electrum addresses. This way you'll have all your Reddcoin secured with your pass-phrase and will only need to keep your initial passphrase backup. Keep in mind that sending all your Reddcoins will reset their coin age and be a minor setback for staking.
Where is my wallet file located?
A default wallet file called default_wallet is created under the /wallets directory when you first run Reddcoin Electrum. On Windows, this folder is located in \Users\YourUserName\AppData\Roaming\Local\Electrum-reddcoin. On all other operating systems, it is located in ~/.electrum-reddcoin.
Can I use Reddcoin Electrum on my phone?
Reddcoin Electrum for mobile devices is not currently available, but is under development and will be ready soon.
