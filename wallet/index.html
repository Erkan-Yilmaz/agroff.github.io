What is Reddcoin Electrum?

Reddcoin Electrum is a fast, secure, and user friendly Reddcoin wallet. Electrum doesn't download the blockchain, so the days of waiting for your wallet to sync are gone. Just open your wallet and start using it. Backups are simplified by a natural language passphrase which can be written down or memorized. With Electrum, a single backup will keep your coins safe forever.

Electrum Features

  • Fast: Your wallet doesn't download the blockchain, so it's ready to use instantly.
  • Forgiving: Your wallet can be easily recovered from your natural language passphrase.
  • Safe: Your private keys don't leave your computer and server data is verified using SPV.
  • Auto Sync: Electrum wallets on all devices stay in sync automatically.
  • Cold Storage: You can have secure offline wallets and still safely spend from an online computer.
  • Compatible: Your private keys can be used with other Reddcoin wallets.

Electrum FAQ

What is the Seed?
Reddcoin Electrum uses a 128-bit random seed to generate your private keys. The seed can be represented as a 12-word pass-phrase which is easy to write down or remember. Regular backups are no longer necessary because all of your addresses can be recovered from a single pass-phrase.
What is the gap limit?
The gap limit is the maximum number of consecutive unused addresses in your deterministic sequence of addresses. Reddcoin Electrum uses a gap limit to stop looking for addresses. This is set to 5 by default, so the client generates new addresses until 5 unused addresses are found.
What does it mean to "freeze" an address?
When you freeze an address, the funds in that address will not be used for sending Reddcoin. You cannot send Reddcoin if you don't have enough funds in non-frozen addresses.
What should I do with my old Reddcoin addresses when switching to Reddcoin Electrum?
The best way to switch to Electrum is to send all your Reddcoin to one of the your Electrum addresses. This way you'll have all your Reddcoin secured with your pass-phrase and will only need to keep your initial passphrase backup. Keep in mind that sending all your Reddcoins will reset their coin age and be a minor setback for staking.
Where is my wallet file located?
A default wallet file called default_wallet is created under the /wallets directory when you first run Reddcoin Electrum. On Windows, this folder is located in \Users\YourUserName\AppData\Roaming\Local\Electrum-reddcoin. On all other operating systems, it is located in ~/.electrum-reddcoin.
Can I use Reddcoin Electrum on my phone?
Reddcoin Electrum for mobile devices is not currently available, but is under development and will be ready soon.

Installing Electrum

Windows Installation

On Windows, simply download and run Reddcoin Electrum

OSX Installation

On OSX, simply download and run Reddcoin Electrum

Linux Installation

To install and run Reddcoin Electrum in Linux, open a shell and run the following commands:

sudo apt-get install python-qt4 python-pip python-dev sudo apt-get install python-slowaes # on Ubuntu 13.10 and higher sudo pip install http://reddwallet.org/download/Reddcoin-Electrum.tar.gz reddcoin-electrum
